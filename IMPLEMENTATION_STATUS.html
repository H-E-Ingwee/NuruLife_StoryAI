🎯 Getting Started (3 Steps)

Step 1: Start Backend

Open File Explorer and navigate to: storyai-backend/

Double-click: START_BACKEND.bat

✓ Backend will start on http://localhost:8000

Step 2: Verify Backend

curl http://localhost:8000/health

Response should be: {"status": "ok"}

Step 3: Start Frontend

npm run dev

Frontend will open at: http://localhost:5176

That's it! Your full-stack app is now running. You can see the login form in the browser.
